Detailed Description

Interface for the result of a simulation (rollout) in MCTS.

This is a marker interface. Specific games should implement this to carry whatever data they need from a simulation (e.g., winner, final score, etc.). The interpretation of this data into a reward value is done by the Node's interpret_result method.
