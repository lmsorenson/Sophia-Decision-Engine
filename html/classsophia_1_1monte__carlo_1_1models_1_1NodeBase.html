Detailed Description

template<typename TState, typename TChange>
class sophia::monte_carlo::models::NodeBase< TState, TChange >

A template base class for implementing the Node interface.

This class provides a generic implementation of a node, using templates to support various types of states and changes. It manages the state of the node and provides access to a factory for creating related objects in the tree.

Template Parameters
TStateThe type representing the state of the system (e.g., a board configuration).
TChangeThe type representing a change that can be applied to a state (e.g., a move).
