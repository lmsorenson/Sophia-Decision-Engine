Detailed Description

Represents an action that can be taken from a specific state (Node) in the search tree.

An action is a transition from a source node to a target node. It represents a move, a decision, or any other change in the state of the system being modeled.

double Action::upper_confidence_bound ( int  c) const

Calculates the Upper Confidence Bound (UCB1) for this action.

UCB1 is a formula used to balance exploration and exploitation in MCTS. It's calculated based on the action's value and the number of times it has been visited.
