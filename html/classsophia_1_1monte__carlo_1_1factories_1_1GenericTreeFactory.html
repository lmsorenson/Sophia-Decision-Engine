Detailed Description

template<typename TState, typename TChange, typename TNode, typename TAction, typename TKey = std::string>
class sophia::monte_carlo::factories::GenericTreeFactory< TState, TChange, TNode, TAction, TKey >

A generic implementation of TreeFactoryBase that creates nodes and actions of specified types.

This factory assumes that TNode and TAction have constructors that match the standard signatures used in the Monte Carlo framework.

Template Parameters
TStateThe state type.
TChangeThe change type.
TNodeThe concrete Node type to create.
TActionThe concrete Action type to create.
TKeyThe type used to identify and select rollout strategies.
