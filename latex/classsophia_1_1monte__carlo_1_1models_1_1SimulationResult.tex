\doxysubsection{Detailed Description}
Interface for the result of a simulation (rollout) in MCTS. 

This is a marker interface. Specific games should implement this to carry whatever data they need from a simulation (e.\+g., winner, final score, etc.). The interpretation of this data into a reward value is done by the \doxylink{classsophia_1_1monte__carlo_1_1models_1_1Node}{Node}\textquotesingle{}s interpret\+\_\+result method.
