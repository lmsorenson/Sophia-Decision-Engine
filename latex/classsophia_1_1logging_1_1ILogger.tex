\doxysubsection{Detailed Description}
An interface for a generic, type-\/safe, and performant logger. 

This interface uses std\+::format for type-\/safe message formatting and defers the formatting until after the log level has been checked, providing performance benefits.

Concrete implementations (e.\+g., \doxylink{classsophia_1_1logging_1_1ConsoleLogger}{Console\+Logger}, \doxylink{classsophia_1_1logging_1_1FileLogger}{File\+Logger}) must implement the private virtual {\ttfamily log} method.
