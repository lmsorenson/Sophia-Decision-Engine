\subsubsection*{template$<$typename TState, typename TChange, typename TNode, typename TAction, typename TKey = std\+::string$>$\newline
class sophia\+::monte\+\_\+carlo\+::factories\+::\+Generic\+Tree\+Factory$<$ TState, TChange, TNode, TAction, TKey $>$}
A generic implementation of \doxylink{classsophia_1_1monte__carlo_1_1factories_1_1TreeFactoryBase}{Tree\+Factory\+Base} that creates nodes and actions of specified types. 

This factory assumes that TNode and TAction have constructors that match the standard signatures used in the Monte Carlo framework.

\hline
